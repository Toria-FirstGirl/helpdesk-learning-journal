
# CMD Basics for Help Desk

## Overview
This document explains common Command Prompt (CMD) commands practiced for basic troubleshooting and system checks in help desk environments.

---

## Why CMD Is Important in Help Desk
CMD allows support technicians to quickly diagnose network, user, and system issues without relying on graphical tools.

---

## Commands Practiced

### ipconfig
Used to check IP address and network configuration.

### ping
Used to test network connectivity to another device or website.

### net user
Used to view and manage local user accounts.
